## Key Technical Details

### Preprocessing (01_preprocess)
- Custom pure-Python slicing (no COCO conversion needed)
- Each ~3000×4000 image → ~48 tiles (640×640, stride=512)
- BBox transform: absolute coords → clip to tile → re-normalize
- Filters out: heavily clipped bboxes (<10% area remaining), micro-bboxes (<16 px²), empty tiles

### Training (02_train)
- **Model**: `yolo26m.pt` (YOLO26 Medium)
- **Optimizer**: MuSGD (SGD + Muon hybrid, native to YOLO26)
- **AMP**: Enabled (FP16, ~10 GB VRAM at batch=16)
- **Augmentations**: mosaic=1.0, mixup=0.15, copy_paste=0.1, HSV jitter, rotation, scale, erasing
- **Early stopping**: patience=30, max 150 epochs
- **Dataset caching**: RAM cache enabled (fits 32 GB)

### Inference (03_inference)
- SAHI `get_sliced_prediction` with 640×640 sliding window, 20% overlap
- NMS merge at IoU=0.5 for overlapping tile predictions
- VOC-style 11-point AP calculation for mAP₅₀
- Annotated images saved with green bboxes + confidence scores
- Full metrics exported to JSON

### Plot Curves (04_plot_curves)
- Parses Ultralytics `results.csv` (auto-generated during training)
- 2×3 grid: Box Loss | Cls Loss | DFL Loss | mAP | Precision/Recall | LR Schedule
- Best mAP₅₀ epoch annotated with marker + arrow
- Saved at 200 DPI to `outputs/training_curves.png`
